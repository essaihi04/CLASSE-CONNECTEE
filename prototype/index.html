⚙️ Avatar 3D animé (iframe)

Crée ton avatar 3D animé sur un service externe, puis colle ici son lien ou son code <iframe>. L'avatar s'affichera dans le tableau.

Où créer un avatar 3D animé qui donne un iframe :
• Spline (spline.design) — anime un personnage 3D, « Export → Public URL / iframe ». Gratuit.
• Sketchfab (sketchfab.com) — un modèle 3D animé, bouton « Embed » → copie l'iframe (active autostart + animation).
• Ready Player Me + un viewer 3D (anime via Mixamo).
Astuce : la voix reste assurée par l'app (synthèse vocale FR). L'iframe apporte le mouvement et les gestes.
